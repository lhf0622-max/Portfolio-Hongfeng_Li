Mini Renderer

COMPUTATIONAL DESIGN

A mini renderer I developed as a WebGL practice to explore core computer graphics concepts. Implemented PCSS for soft shadows, real-time ray tracing for basic reflections, and spherical harmonics for ambient lighting. This project serves as a hands-on study of rendering pipelines, focusing on the balance between visual realism and computational efficiency in real-time environments.

Physical Simulation

COMPUTATIONAL DESIGN

This is a series of lightweight physics engine projects. I used Unity scripts to explore core simulation and animation techniques in computer graphics, implementing fluid and cloth simulations as well as the Finite Element Method (FEM) for elastic material deformation.

Additionally, I integrated a character animation system supporting skinning and rigging for MoCap data, utilizing animation blending for seamless state transitions.
